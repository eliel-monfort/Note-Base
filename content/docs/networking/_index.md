## Introduction

How exactly do computers, phones, or any electronic devices talk to each other?  
How can we access websites that anyone else can reach?  
How is it possible to play a computer game with friends when you're not in the same room?

All of this is what **networking** is about.  
In short, it's how devices connect so they can share information and communicate.  
Networking is everywhere - at home, at school, in offices, big companies and in huge data centers.

## Scope

Here is what I cover:

### Included:

- How devices connect and communicate.
- Common network types.
- Basic rule the devices most use for communicate - called Protocols.
- Useful command and tools.

### Not Included:

- How exactly the hardware in the devices works to communicate.
- Advanced hacking and security tricks.
